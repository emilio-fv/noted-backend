# Noted Backend

Current API endpoints:
* Auth Features
    - Register user
    - Login user
    - Logout user
    - Refresh access token
* Music Features
    - Get Spotify access token
    - Get featured albums
    - Query Spotify
    - Get artist's data
    - Get album's data
 * Reviews Features
    - Create review
    - Get reviews by logged in user
    - Get reviews by users the logged in user is following
    - Get reviews by artist id
    - Get reviews by album id
    - Get reviews by username
    - Edit review
    - Like review
    - Unlike review
    - Delete review 
* Connect Features
    - Quey users
    - Get user profile data by username
    - Follow user
    - Unfollow user
